# 贪吃蛇 Atelier

这是一个单文件网页游戏。主入口是 `index.html`，现代浏览器直接打开就能玩。

## 本机打开

```bash
open index.html
```

## 发给朋友

把整个 `Agents` 文件夹压缩后发给朋友，朋友解压后双击 `index.html` 即可。

## 变成任何人都能点开的链接

最快方式是 Netlify Drop：

1. 打开 `https://app.netlify.com/drop`
2. 把这个文件夹拖进去
3. 等它生成网址，把网址发给朋友

也可以发布到 GitHub Pages、Vercel 或任何静态网站托管服务。只要服务能识别根目录的 `index.html`，就能直接打开游戏。
